# HectX Report Rendering

## Render To HTML

1. Install dependencies:

```
npm install
```

2. Render the report:

```
node scripts/render-report.mjs docs/hectx-report/hectx-implementation-report.md docs/hectx-report/hectx-implementation-report.html
```

The output HTML is styled in a Wall Street Journal-inspired layout using Tailwind via CDN.
