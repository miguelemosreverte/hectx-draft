**Abstract**
Fee application and mint calculations implemented in code.

**Body**
I. Fee Computation
I.a Subscription fee and conversion fee are applied using basis points.
I.b Elastic fee applies only when active and market price exceeds NAV-per-token.

II. Calculation Logic
```
let subFee = amount * fee.subscriptionFeeBps / 10000.0
let convFee = amount * fee.conversionFeeBps / 10000.0
let premium = if marketPrice > navPerToken then marketPrice - navPerToken else 0.0
let elasticFee = if policy.elasticFeeActive then premium * policy.elasticFeeBps / 10000.0 else 0.0
let netAmount = amount - subFee - convFee - elasticFee
let mintedAmount = netAmount / navPerToken
```
