**Abstract**
Concrete JSON API usage for admin, minting, and transfer workflows with commands to fetch required contract IDs.

**Body**
I. Admin Bootstrap
```bash
cd hectx-services
npm install
npm run build
export JSON_API_URL=http://localhost:2975/v1
export ADMIN_PARTY=Hecto
npm run admin
```

II. Fetch Registry And Transfer Factory IDs
```bash
export JSON_API_URL=http://localhost:2975/v1
curl -sS -X POST "$JSON_API_URL/query" \
  -H "Content-Type: application/json" \
  -d '{"templateId":"HectX.Registry:HectXRegistry","query":{}}' | jq -r '.[0].contractId'

curl -sS -X POST "$JSON_API_URL/query" \
  -H "Content-Type: application/json" \
  -d '{"templateId":"HectX.Transfers:HectXTransferFactory","query":{}}' | jq -r '.[0].contractId'
```

III. Mint
```bash
export JSON_API_URL=http://localhost:2975/v1
export ADMIN_PARTY=Hecto
export INVESTOR_PARTY=Alice
export REGISTRY_CID=$(curl -sS -X POST "$JSON_API_URL/query" -H "Content-Type: application/json" -d '{"templateId":"HectX.Registry:HectXRegistry","query":{}}' | jq -r '.[0].contractId')
export MARKET_PRICE=1.0
npm run mint
```

IV. Transfer
```bash
export JSON_API_URL=http://localhost:2975/v1
export ADMIN_PARTY=Hecto
export SENDER_PARTY=Alice
export RECEIVER_PARTY=Bob
export HOLDING_CID=$(curl -sS -X POST "$JSON_API_URL/query" -H "Content-Type: application/json" -d '{"templateId":"HectX.Holding:HectXHolding","query":{}}' | jq -r '.[0].contractId')
export TRANSFER_FACTORY_CID=$(curl -sS -X POST "$JSON_API_URL/query" -H "Content-Type: application/json" -d '{"templateId":"HectX.Transfers:HectXTransferFactory","query":{}}' | jq -r '.[0].contractId')
export AMOUNT=100.0
npm run transfer
```
