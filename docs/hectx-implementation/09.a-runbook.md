**Abstract**
Step-by-step runbook to execute the working prototype.

**Body**
I. Start LocalNet
```bash
./scripts/localnet-up.sh
```

II. Build Daml
```bash
./scripts/build-daml.sh
```

III. Run Daml Tests
```bash
./scripts/test-daml.sh
```

IV. Build Services
```bash
cd hectx-services
npm install
npm run build
```

V. Bootstrap Policies And Registry
```bash
JSON_API_URL=http://localhost:2975/v1 \
ADMIN_PARTY=Hecto \
npm run admin
```
