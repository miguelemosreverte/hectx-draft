**Abstract**
Concrete transfer flow as implemented by `HectXTransferFactory`.

**Body**
I. Transfer Steps
I.a Validate timing and instrument ID.
I.b Enforce eligibility and wallet approvals.
I.c Consume input holdings and create receiver holding and sender change.

II. Code Reference
```
transferFactory_transferImpl _ arg = do
  now <- getTime
  require "requestedAt must be in the past" (transfer.requestedAt <= now)
  require "executeBefore must be in the future" (transfer.executeBefore > now)
  senderP <- fetchByKey @Participant transfer.sender
  receiverP <- fetchByKey @Participant transfer.receiver
  require "sender not eligible" (senderP.status == Eligible)
  holdings <- forA transfer.inputHoldingCids $ \ifaceCid -> do
    let hCid : ContractId HectXHolding = fromInterfaceContractId ifaceCid
    fetch hCid
```
