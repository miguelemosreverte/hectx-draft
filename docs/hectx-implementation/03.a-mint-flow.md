**Abstract**
Concrete mint flow as implemented by `MintRequest.ApproveMint`.

**Body**
I. Mint Steps
I.a Fetch registry, fee schedule, mint policy, NAV snapshot, and supply.
I.b Validate eligibility and wallet approval.
I.c Compute NAV-per-token and fees.
I.d Create holding and update supply.

II. Code Reference
```
choice ApproveMint : ContractId MintReceipt
  with registryCid : ContractId HectXRegistry; marketPrice : Decimal
  controller admin
  do
    navSnap <- fetchByKey @NAVSnapshot admin
    supply <- fetchByKey @Supply admin
    let navPerToken = if supply.totalSupply == 0.0 then 1.0 else navSnap.nav / supply.totalSupply
    let mintedAmount = netAmount / navPerToken
    archive supply
    _ <- create Supply with admin; totalSupply = supply.totalSupply + mintedAmount
    holdingCid <- create HectXHolding with ...
```
