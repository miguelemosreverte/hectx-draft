**Abstract**
Concrete eligibility and wallet approval records.

**Body**
I. Participant Template
```
template Participant
  with
    admin : Party
    party : Party
    jurisdiction : Text
    isAccredited : Bool
    status : EligibilityStatus
  where
    signatory admin
    observer party
    key party : Party
    maintainer admin
```

II. Wallet Approval Template
```
template WalletApproval
  with
    admin : Party
    owner : Party
    active : Bool
  where
    signatory admin, owner
    key owner : Party
    maintainer admin
```
