**Abstract**
Runtime wiring between Daml contracts, LocalNet, and JSON API clients.

**Body**
I. Runtime Sequence
I.a Start LocalNet: `scripts/localnet-up.sh`.
I.b Build and deploy Daml package: `scripts/build-daml.sh` and upload DAR to participant.
I.c Bootstrap registry and policies: `hectx-services/src/admin.ts`.
I.d Mint and transfer using JSON API clients.

II. Network Endpoints
I.a JSON API endpoint is provided by the Canton participant (LocalNet exposes ports based on `env/common.env`).
I.b Swagger UI is exposed on `SWAGGER_UI_PORT` (default 9090).
