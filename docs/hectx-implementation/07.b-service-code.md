**Abstract**
JSON API client used by the admin, mint, and transfer scripts.

**Body**
I. JSON API Client
```ts
export const create = async (config, templateId, payload) => {
  const res = await fetch(`${config.baseUrl}/v1/create`, {
    method: "POST",
    headers: headers(config.token),
    body: JSON.stringify({ templateId, payload }),
  });
  if (!res.ok) throw new Error(...);
  return res.json();
};
```
