**Abstract**
Jurisdiction policy data structures used in the prototype.

**Body**
I. Restricted Rule
```
data RestrictedRule = RestrictedRule with
    description : Text
    minNetWorth : Optional Decimal
    minAssets : Optional Decimal
    minIncome : Optional Decimal
```

II. Policy Contract
```
template EligibilityPolicy
  with
    admin : Party
    prohibitedJurisdictions : [Text]
    restrictedJurisdictions : TextMap.TextMap RestrictedRule
  where
    signatory admin
```
