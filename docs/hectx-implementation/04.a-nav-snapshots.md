**Abstract**
NAV data structures used by minting.

**Body**
I. NAV Snapshot Template
```
module HectX.NAV where

template NAVSnapshot
  with
    admin : Party
    nav : Decimal
    gav : Decimal
    reserves : Decimal
    liabilities : Decimal
    accruedFees : Decimal
    timestamp : Time
  where
    signatory admin
    key admin : Party
    maintainer admin
```

II. Supply Template
```
module HectX.NAV where

template Supply
  with
    admin : Party
    totalSupply : Decimal
  where
    signatory admin
    key admin : Party
    maintainer admin
```
