**Abstract**
This document set is a high-level, implementation-oriented summary of the HectX documentation. It preserves the product’s NAV-based mechanics, compliance perimeter, valuation methodology, and operational constraints, and organizes them into an implementation-ready structure. It is not a legal document and does not replace Hecto’s official terms.

**Body**
I. Purpose And Scope
I.a Translate HectX documentation into concrete requirements for product design and system behavior.
I.b Capture the minimum set of rules needed to implement onboarding, minting, valuation, transferability, and governance.

II. System Boundary
II.a In scope: onboarding, eligibility gates, pricing inputs, NAV computation, minting, transfers, and reporting.
II.b Out of scope: portfolio manager discretion, external market pricing, and final legal agreements.

III. Evidence Discipline
III.a Every requirement maps to HectX documentation pages referenced below.
III.b External references are used only where the HectX docs explicitly cite them.
